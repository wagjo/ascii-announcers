# ASCII Announcers

ASCII announcers for cowsay like announcements

## Installation

Using pip:

```sh
pip install ascii-announcers
```

## Usage

```python
from ascii_announcers import announcers

print(list(announcers.keys()))

print(announcers["dog"])
```
